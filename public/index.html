Frequently Asked Questions

Everything you need to know about the MKR1000 Plant Monitoring System

What is the Arduino MKR1000 and why use it for plant monitoring?

The Arduino MKR1000 is a powerful microcontroller board with integrated WiFi connectivity. It features a 32-bit ARM Cortex-M0+ processor and is ideal for IoT projects. Its low power consumption, built-in WiFi, and crypto chip make it perfect for secure, always-on plant monitoring systems that can be accessed remotely.

How does Johnny-Five.io work with Arduino?

Johnny-Five is a JavaScript robotics and IoT platform that runs on Node.js. It communicates with the Arduino MKR1000 using the Firmata protocol, allowing you to control hardware using JavaScript instead of traditional Arduino C++. This enables rapid development, web integration, and access to the vast Node.js ecosystem.

What sensors are included in the system?

The system monitors three critical parameters: (1) Voltage sensor (11.5-14.5V range) for power supply monitoring, (2) Current sensor (0.3-2.5A range) for electrical consumption tracking, and (3) Soil moisture sensor (0-100% range) for precise irrigation control. All sensors provide real-time data updated every second.

Can I access the dashboard remotely?

Yes! The web-based dashboard is accessible from any device on your local network. The Arduino MKR1000 connects to your WiFi network, and the Node.js server provides a responsive web interface. For remote access from outside your network, you can configure port forwarding or use services like ngrok.

How do I set up the system?

Setup involves three main steps: (1) Upload the Standard Firmata WiFi sketch to your Arduino MKR1000, (2) Connect your sensors to the appropriate pins, (3) Install Node.js dependencies and run the server application. Detailed documentation is available on our GitHub repository with step-by-step instructions.

Can I expand the system with more sensors?

Absolutely! The modular architecture allows you to add temperature sensors, pH sensors, light sensors, or any other analog/digital sensors. The Johnny-Five library supports a wide range of sensors, and the dashboard can be customized to display additional data streams.

Is the system suitable for commercial agriculture?

While designed as an educational and hobbyist platform, the system's professional-grade components and reliable WiFi connectivity make it suitable for small to medium commercial applications. Many users have successfully deployed multiple units in greenhouse operations, vertical farms, and research facilities.

What makes this better than other plant monitoring solutions?

Our system combines enterprise-grade hardware (Arduino MKR1000) with modern web technologies (Node.js, Highcharts). Unlike closed commercial systems, it's completely open-source, customizable, and extensible. The use of Standard Firmata and Johnny-Five provides unmatched flexibility for integration with other systems and sensors.
